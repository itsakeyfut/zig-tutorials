# 04. 関数定義

このチュートリアルでは、Zigにおける関数定義の様々なパターンについて学びます。

## サンプルファイル

### 1. src/main.zig - 基本的な関数

基本的な関数の定義と使用方法を学びます。

**実行方法:**
```bash
zig build run
```

**学習内容:**
- `fn` キーワードでの関数定義
- 戻り値の型指定
- `pub` による公開関数
- 関数の呼び出し

### 2. src/multiple_return.zig - 複数の戻り値（構造体を返す）

構造体を使って複数の値を返す方法を学びます。

**実行方法:**
```bash
zig run src/multiple_return.zig
```

**学習内容:**
- 構造体を戻り値として使用
- `@divTrunc` と `@rem` の使い方
- 構造体のフィールドアクセス

### 3. src/arguments.zig - 引数の扱い

ポインタやスライスなど、様々な引数の渡し方を学びます。

**実行方法:**
```bash
zig run src/arguments.zig
```

**学習内容:**
- ポインタ引数の使用
- スライスの扱い
- 参照渡しと値渡し

## 学習ポイント

- Zigでは `fn` で関数を定義
- `pub` で関数を公開
- 戻り値の型は `->` ではなく直接指定
- `return` は必須（Rustのような暗黙的な戻り値なし）

## よくあるコンパイルエラー

```
error: expected type 'i32', found 'void'
```

**原因:** 戻り値の型と実際の戻り値が不一致

**対処:** `return` 文を追加または型を修正
